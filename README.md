# Data Cleaning, Analysis & Dashboarding using Excel

## Project Overview
This project demonstrates how Excel can be used to clean, analyze, and visualize data by transforming raw data into meaningful insights through structured analysis and interactive dashboards.

## Tools
- Microsoft Excel

## Data Cleaning Steps
- Removed duplicate records to ensure data accuracy  
- Standardized data formats and corrected inconsistencies  
- Handled null and blank values appropriately  
- Cleaned and structured the dataset for analysis  

## Data Analysis
- Created Pivot Tables to summarize and analyze key metrics  
- Applied filters and calculated fields to answer business questions  
- Identified trends and patterns within the data  

## Dashboard
- Built an interactive dashboard using Excel charts  
- Visualized key insights through graphs and KPIs  
- Designed the dashboard for clear and easy interpretation  

## Outcome
The final output is a clean dataset, structured analysis using Pivot Tables, and a dynamic Excel dashboard that supports data-driven decision-making.
